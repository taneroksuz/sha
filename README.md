# SHA
Hardware and software implementation of SHA algorithm. This algorithm is implemented in Verilog and C++.

## Hardware Implementation
This algorithm is implemented in finite state machine.

| Algorithm  | SHA256 | SHA512 |
|:-----------|:------:|:------:|
| LUT        | 3172   | 7896   |
| FF         | 2617   | 6224   |
| BRAM       | 0      | 1      |
